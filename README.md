In most applications, buttons are used in many places with different styles and behaviors.
Instead of repeating button code, this project provides a single generic button component that can be customized using props.

Key Features

Reusable button component
Type-safe props using TypeScript
Styled with Tailwind CSS
Supports different variants (primary, secondary, danger)
Supports different sizes (small, medium, large)
Handles disabled and loading states
Easy to extend and maintain

Tech Stack

React
TypeScript
Tailwind CSS
